
# WhatsAppUI

This is an Application replicating whatsApp's UI using Flutter


## Run Locally

Clone the project

```bash
  git clone https://github.com/godsmith53/WhatsAppUI
```

Go to the project directory

```bash
  cd whatsappui
```

Install dependencies

```bash
  flutter pub get
```

Start the App

```bash
  flutter run
```
